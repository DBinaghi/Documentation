.. php:function:: file_markup(File $files, array $props, array $wrapperAttributes)

    Return HTML for a set of files.
    
    :param File $files: A file record or an array of File records to display.
    :param array $props: Properties to customize display for different file types.
    :param array $wrapperAttributes: Attributes HTML attributes for the div that wraps each displayed file. If empty or null, this will not wrap the displayed file in a div.
    :returns: string HTML
